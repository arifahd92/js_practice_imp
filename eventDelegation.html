Event Propagation Recap Event propagation in the DOM consists of three phases: Capturing Phase: The event travels from the root of the DOM tree (e.g., document) to the target element. Target Phase: The event reaches the target element where it was triggered. Bubbling Phase: The event travels back up from the target element to the root of the DOM tree. By default, most event listeners operate in the bubbling phase, but we can use event capturing by setting the capture option to true.
